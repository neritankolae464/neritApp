## Usage

### Size

Use the `size` prop to change the icon size, adhering to an 8px grid (e.g., 16px, 24px, 32px).

<Canvas>
  <Story id="components-ui-icon--size-story" />
</Canvas>

### Color

Use the `color` prop to set the icon color. Accepts any CSS value; prefer design token CSS variables.

<Canvas>
  <Story id="components-ui-icon--color" />
</Canvas>

### AriaLabel

Use `ariaLabel` for icons without text for accessibility (e.g., back buttons).

<Canvas>
  <Story id="components-ui-icon--aria-label" />
</Canvas>

Will render:

```html
<svg width="24" height="24" fill="currentColor" aria-label="back"
     xmlns="http://www.w3.org/2000/svg" viewBox="0 0 512 512">
   <path d="m335 113c8 8 8 22..."></path>
</svg>
```

### ClassName

Add additional classes via `className` for layout or positioning.
